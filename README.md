# DSA Projects

This repository contains a collection of Data Structures and Algorithms (DSA) projects, each designed to demonstrate and visualize core concepts in computer science. Below is an overview of each project folder:

## 1. algo_visualizer
- **Purpose:** Visualizes various algorithms, such as pathfinding and sorting, to help users understand their step-by-step execution.
- **Key Files:** `main.py`, `algorithms.py`, `visualizer.py`, `maze.py`, `ui_components.py`
- **Features:** Interactive UI for algorithm visualization.

## 2. cryptography
- **Purpose:** Implements basic cryptographic algorithms and concepts.
- **Key Files:** `main.py`, `app.py`
- **Features:** Demonstrates encryption, decryption, and related cryptographic operations.

## 3. dijkrsta_algo
- **Purpose:** Visualizes Dijkstraâ€™s algorithm for shortest path finding.
- **Key Files:** `main.py`, `main.html`, `algo.js`, `script.js`
- **Features:** Web-based and Python implementations; includes documentation.

## 4. sorting_algo
- **Purpose:** Visualizes and implements various sorting algorithms.
- **Key Files:** `main.py`, `app.py`, `sorting_algorithm.py`, `visualizer.py`
- **Algorithms:** Bubble Sort, Insertion Sort, Merge Sort, Quick Sort, Selection Sort (see `algorithms/` folder).
- **Features:** Audio engine for interactive feedback.

## 5. sudoku_game
- **Purpose:** Sudoku game with solver and visualizer.
- **Key Files:** `main.py`, `sudoku_board.py`, `sudoku_solver.py`, `sudoku_visualizer.py`, `puzzles.py`
- **Features:** Playable Sudoku puzzles, automatic solving, and visualization.

## 6. trie_llm
- **Purpose:** Demonstrates Trie data structure and its applications.
- **Key Files:** `main.py`, `trie.py`, `visualize_trie.py`, `schema.py`, `demo_explained.py`
- **Features:** Trie visualization and schema-based demos.
