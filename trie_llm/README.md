# Trie LLM Demo

A Python project to demonstrate trie data structures and their visualization, with examples and schema support.

## Features
- Trie data structure implementation
- Visualize trie operations
- Demo scripts for understanding usage
- Schema for structured data

## Requirements
- Python 3.8+

## Getting Started
1. Clone this repository or copy the folder.
2. (Optional) Create a `.env` file for environment variables (see below).
3. Run the main demo:
   ```bash
   python main.py
   ```

## File Structure
- `main.py` — Entry point for trie demo
- `trie.py` — Trie data structure implementation
- `visualize_trie.py` — Visualization logic
- `demo_explained.py` — Explained demo script
- `reel_demo.py` — Additional demo
- `schema.py` — Data schema

## .env Example
```
# Example environment variables
TRIE_DEMO_MODE=basic
LOG_LEVEL=info
```

## How to Use
- Run the demo scripts to see how tries work and are visualized.
- Customize `.env` for different modes or logging.
